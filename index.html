🔥 Firebase Setup
Connect your Firebase Realtime Database for real-time cross-device sync. Only required once — saved in your browser.
1. Go to console.firebase.google.com → Project Settings → Web app
2. Set Realtime Database rules:
{ "rules": { ".read": true, ".write": true } }

🔥 Firebase Setup
Connect your Firebase Realtime Database for cross-device live sync. This only needs to be done once — settings are saved in your browser.
1. Go to console.firebase.google.com → your project → Project Settings → Web app config
2. Paste your databaseURL below (looks like https://YOUR-PROJECT-default-rtdb.firebaseio.com)
⚠️ Firebase Rules: Set your Realtime Database rules to allow read/write (for testing):
{ "rules": { ".read": true, ".write": true } }

🔐 Referee Passwords

Set unique passwords for each referee. They enter this on their phone to unlock their court's control panel. Changes sync to Firebase instantly.

How it works:
📱 Ref A opens the URL on their phone → clicks "Ref A" → enters their password
📱 Ref B opens the URL on their phone → clicks "Ref B" → enters their password
📺 Display screen is shown on big screen/projector (use the ⛶ fullscreen button)

All changes appear on the display instantly via Firebase.
